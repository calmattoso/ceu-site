This is the online tutorial for the Céu programming language.

We assume some knowledge in C, as most of the basic functionality of Céu derives from it (constants, operators, declarations, etc).

Feel free to modify the examples in the Code and Input panels on the right.

To run a program, click the Run button on the top-right of the page. The results appear in the panel on the bottom.

To advance to the next lesson, click the Next button above.
