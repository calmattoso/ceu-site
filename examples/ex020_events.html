Céu interacts with the environment by awaiting external input events, such as MY_EVT in this example.

The await statement accounts for the reactive nature of Céu.

An input event in Céu represents an event in the underlying platform, aka environment. The availability of external events depends on the platform in use.

Céu offers a number of environments for the development of real applications:

  • SDL for games and graphical applications.
  • Arduino for embedded systems.
  • libuv for network applications.

In this tutorial, we simulate the occurrence of abstract inputs such as MY_EVT.
