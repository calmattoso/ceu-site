Céu source files can be optionally preprocessed with m4, providing a similar functionality to the C preprocessor.

Moreover, m4 can also be used to substitute blocks of code, providing a powerful mechanism for creating abstractions in Céu.

This example defines the macro AWAIT_COND that loops until the value of an event satisfies a condition. The event name ($1), condition operator ($2), and expected value ($3) should be passed as arguments to the macro.

The two occurrences AWAIT_COND in the source code are expanded to the macro definition.

In m4, command define is used to define a new macro, and symbols ` and ' are used to enclose tokens that contain spaces and special characters.

In Céu, the token @ is ignored by the parser and is used to highlight the use of a macro.

High-level abstractions can be created with m4!
