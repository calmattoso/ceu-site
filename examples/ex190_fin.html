Some C functions produce a side effect that requires a further call to destroy that effect. For example, a call to malloc will allocate memory that should be destroyed in the future. In Céu, any variable that is assigned a pointer from a function call requires a finalization code. Internal events are used as a signaling mechanism among trails.

A program can await for changes on internal events, which can be signaled through the emit statement.

The emitted value can be captured by the corresponding awaiting trails.
