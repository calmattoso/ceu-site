Céu is a concurrent and reactive programming language that offers a safer and higher-level alternative to C.


This is how Céu looks like:

input int KEY;
par/or do
    every 1s do
        _printf("Hello World!\n");
    end
with
    await KEY;
end
(Prints the "Hello World!" message every second, terminating on a key press.)

See more introductory videos in this link .

Features

Céu supports synchronous concurrency with shared memory and deterministic execution. Céu has a small memory footprint and integrates seamlessly with C.
