#### This is what <span class="ceu-text">C&eacute;u</span> looks like:

```ceu
input int KEY;  
par/or do  
  every 1s do  
    _printf("Hello World!\n");  
  end  
with  
  await KEY;  
end
```

<i><small>(Prints the "Hello World!" message every second, terminating on a 
key press.)</small></i>
